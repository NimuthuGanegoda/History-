# Markdown Content

This directory contains all historical content in Markdown format - a modern, lightweight markup language that's easy to read, edit, and version control.

## Structure

```
markdown-output/
├── kings/          # Individual king biographies (180+ files)
└── kingdoms/       # Kingdom overviews (15 files)
```

## Format

Each file includes:
- **Frontmatter** (YAML metadata at the top)
- **Title and headers** in Markdown syntax
- **Clean, readable content** without HTML tags

## Example

```markdown
---
title: "King Vijaya"
reign: "543–505 BCE"
kingdom: "Tambapanni"
layout: page
---

# King Vijaya

## Biography

King Vijaya is the legendary first king of Sri Lanka...
```

## Benefits of Markdown

✅ **Easy to edit** - No HTML knowledge required  
✅ **Version control friendly** - Clean diffs in git  
✅ **Portable** - Can be converted to HTML, PDF, etc.  
✅ **Modern** - Used by GitHub, documentation sites, static generators  
✅ **Future-proof** - Plain text format

## Usage

These Markdown files can be:
- Converted to HTML with static site generators (Jekyll, Hugo, Gatsby)
- Displayed directly on GitHub
- Imported into modern CMS systems
- Used with documentation platforms
